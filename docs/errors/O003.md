# Error O003: Mutable Borrow Conflict

This error occurs when trying to mutably borrow a variable when it is already mutably or shared borrowed.

## Incorrect Code Example

```arandu
let x = Data {}
let r1 = &mut x
let r2 = &mut x // Conflict
```

## Semantic Explanation

To ensure data race safety and memory consistency, you can only have a single active mutable reference at any given time.

## How to Fix

Ensure borrow lifetimes do not overlap:

```arandu
let x = Data {}
{
    let r1 = &mut x
}
let r2 = &mut x // Ok
```
