# Error N005: Value Used as a Type

This error occurs when using a variable or value in a position where a type is expected.

## Incorrect Code Example

```arandu
func main() {
    let x = 10
    y x = 5
}
```

## Semantic Explanation

Type annotations must reference type symbols, not variables or runtime values.

## How to Fix

Use a valid type name in annotations:

```arandu
func main() {
    let x = 10
    y int = 5
}
```
