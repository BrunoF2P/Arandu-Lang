# Error G001: Generic Instantiation Cycle

This error occurs when a recursive generic instantiation pattern creates an infinite monomorphization loop.

## Incorrect Code Example

```arandu
struct Box[T] {
    inner Box[Box[T]]
}
```

## Semantic Explanation

Arandu compiles generics via monomorphization (generating concrete code for each type). Infinite type structures would cause infinite compiler loops.

## How to Fix

Avoid structures that lead to infinite generic parameter growth.
