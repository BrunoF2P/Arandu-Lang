# Error T032: Invalid `await` Operand

The `await` operator can only be applied to a `Coroutine<T>` value. Applying
`await` to any other type is a type error.

## Incorrect Code Example

```arandu
func main(): int {
    let n = 1
    return await n
}
```

## Correct Code Example

```arandu
async func answer(): int {
    return 42
}

func main(): int {
    return await answer()
}
```

## Notes

- `async func` bodies produce `Coroutine[T]` at call sites.
- Prefer awaiting a coroutine (or an async call) rather than a plain value.
