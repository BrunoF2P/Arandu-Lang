# Error P004: Expected Identifier

This error occurs when the parser expects a variable or type name (identifier) but finds a keyword, operator, or literal instead.

## Incorrect Code Example

```arandu
struct = 10
```

## Semantic Explanation

Keywords are reserved symbols and cannot be used as names for variables, functions, or types.

## How to Fix

Choose a non-reserved identifier name:

```arandu
struct_val = 10
```
