# Error T027: Missing struct fields in initializer

A struct was initialized but one or more declared fields were not provided in the instantiation block.

## Erroneous Code Example

```arandu
struct Point {
    x: i32
    y: i32
}

func main() {
    p = Point { x: 10 } // error: missing fields `y` in struct initializer
}
```

## Semantic Explanation

Every field in a struct must be initialized when instantiating the struct to ensure the allocated memory block is fully defined. Leaving fields uninitialized can lead to undefined behavior or reading garbage data.

## How to Fix

Ensure all fields declared in the struct definition are assigned a value in the struct literal block:

```arandu
struct Point {
    x: i32
    y: i32
}

func main() {
    p = Point { x: 10, y: 20 } // ok!
}
```
