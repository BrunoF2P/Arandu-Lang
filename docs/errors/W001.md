# Error W001: Variable Assigned But Not Used

This warning occurs when a variable is declared and assigned a value but is never read in the scope.

## Incorrect Code Example

```arandu
func main() {
    x = 10 // Warning: x is never used
}
```

## Semantic Explanation

Unused variables clutter the code and might indicate incomplete logic or programming mistakes.

## How to Fix

Remove the unused variable, or use it:

```arandu
func main() {
    x = 10
    io.println(x)
}
```
