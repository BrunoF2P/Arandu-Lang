# Error T021: Method Requires Self Receiver

This error occurs when trying to call an instance method as a static function without passing the self receiver.

## Incorrect Code Example

```arandu
struct User {}
func User.greet(self) {}

func main() {
    User.greet()
}
```

## Semantic Explanation

Instance methods require an active instance of the type (the `self` receiver) to bind the call.

## How to Fix

Call the method on an instance or pass the receiver:

```arandu
struct User {}
func User.greet(self) {}

func main() {
    u = User {}
    u.greet()
}
```
