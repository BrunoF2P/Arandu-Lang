# Error L001: Lowering Unresolved Symbol

This error occurs when lowering references a symbol that was not successfully resolved in name resolution.

## Incorrect Code Example

```arandu
// Occurs internally during AST -> HIR conversion
```

## Semantic Explanation

Lowering requires resolved AST nodes. If name resolution failed to link a node to its symbol definition, lowering cannot run.

## How to Fix

Resolve all name resolution errors first.
