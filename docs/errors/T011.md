# Error T011: Generic Constraint Not Satisfied

This error occurs when instantiating a generic function or type with a type parameter that does not satisfy its generic constraints.

## Incorrect Code Example

```arandu
interface Show {}
struct Box[T] where T: Show {}
struct User {}

func main() {
    b = Box[User] {}
}
```

## Semantic Explanation

Generic constraints enforce that type parameters must implement specific interfaces. Instantiating a generic with a type that lacks the interface violates this.

## How to Fix

Pass a type that implements the required interface:

```arandu
interface Show {}
struct Counter {}
func Counter.show(c Counter) {}

struct Box[T] where T: Show {}
func main() {
    // Assuming Counter satisfies Show
    b = Box[Counter] {}
}
```
