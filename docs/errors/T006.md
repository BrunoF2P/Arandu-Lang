# Error T006: Type Not Nullable

This error occurs when assigning `null` to a variable that is not explicitly optional or nullable.

## Incorrect Code Example

```arandu
func main() {
    let x int = null
}
```

## Semantic Explanation

By default, values in Arandu cannot be null. To allow null, you must use pointer types or optional types.

## How to Fix

Make the type nullable by declaring it as a pointer:

```arandu
func main() {
    let x ptr[int] = null
}
```
