# Error T006: Type Not Nullable

This error occurs when assigning `nil` to a variable that is not explicitly optional or nullable.

## Incorrect Code Example

```arandu
func main() {
    x: int = nil
}
```

## Semantic Explanation

By default, values in Arandu cannot be nil. To allow nil, you must use pointer types or optional types.

## How to Fix

Make the type nullable by declaring it as a pointer:

```arandu
func main() {
    x: ptr[int] = nil
}
```
