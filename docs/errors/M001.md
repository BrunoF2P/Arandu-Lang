# Error M001: Unresolved Import

This error occurs when a module or member import path cannot be resolved by the compiler.

## Incorrect Code Example

```arandu
import non_existent_module
```

## Semantic Explanation

During module collection, the compiler searches for imported modules in the project source path and dependencies. If it cannot find the target file or package, it reports unresolved import.

## How to Fix

Verify the import name is correct and the dependency is declared:

```arandu
import io // Correct
```
