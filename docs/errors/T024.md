# Error T024: Non-Exhaustive Match

This error occurs when a `match` expression does not cover all possible patterns of the matched value.

## Incorrect Code Example

```arandu
enum Color { Red, Blue }
func main(c Color) {
    match c {
        Color.Red => 1
    }
}
```

## Semantic Explanation

Match expressions must be exhaustive to guarantee that a branch will always be matched at runtime.

## How to Fix

Cover all variants or add a wildcard fallback arm:

```arandu
enum Color { Red, Blue }
func main(c Color) {
    match c {
        Color.Red => 1,
        Color.Blue => 2
    }
}
```
