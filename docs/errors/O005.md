# Error O005: Double Free

This error occurs when the compiler detects that a single resource would be destroyed or deallocated more than once.

## Incorrect Code Example

```arandu
let x = Data {}
free x
free x // Double free
```

## Semantic Explanation

Deallocating a resource twice leads to memory corruption. The compiler tracks resource destruction paths statically.

## How to Fix

Ensure each resource is deallocated exactly once.
