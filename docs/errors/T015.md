# Error T015: Implicit Widening Not Allowed

This warning occurs when performing implicit conversions that could lead to loss of precision (e.g. i32 to i16).

## Incorrect Code Example

```arandu
let x i16 = 100000 // i32 to i16 conversion
```

## Semantic Explanation

Implicit conversions are restricted to avoid truncation. Explicit casts must be used for potentially lossy conversions.

## How to Fix

Use an explicit cast to confirm the conversion:

```arandu
let x i16 = 100000 as i16
```
