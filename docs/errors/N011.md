# Error N011: Break/Continue Outside Loop

This error occurs when a `break` or `continue` statement is placed outside of a loop scope.

## Incorrect Code Example

```arandu
func main() {
    break
}
```

## Semantic Explanation

Control flow statements `break` and `continue` only make sense inside iterative scopes like `for` and `while` loops.

## How to Fix

Ensure they are nested inside a loop:

```arandu
func main() {
    while true {
        break
    }
}
```
