# Error O010: Escape of Borrowed Value

This error occurs when returning a reference to a local variable allocated on the function stack.

## Incorrect Code Example

```arandu
func get_ref() -> ref int {
    let x = 5
    return &x // Escapes stack!
}
```

## Semantic Explanation

Stack-allocated local variables are destroyed when their function returns. Returning a reference to them would create a dangling pointer.

## How to Fix

Return the value itself, or allocate the memory on the heap:

```arandu
func get_val() -> int {
    let x = 5
    return x
}
```
