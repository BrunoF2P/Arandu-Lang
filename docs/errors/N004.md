# Error N004: Type Used as a Value

This error occurs when using a type name in a position where a value is expected.

## Incorrect Code Example

```arandu
struct User {}
func main(): {
    x =: User
}
```

## Semantic Explanation

Type names exist strictly for compile-time type checking and cannot be evaluated as values at runtime.

## How to Fix

Instantiate the type or use a value:

```arandu
struct User {}
func main(): {
    x = User {}
}
```
