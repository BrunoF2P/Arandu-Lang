# Error W004: Variable Shadowing

This warning occurs when declaring a variable in an inner scope that has the same name as a variable in an outer scope.

## Incorrect Code Example

```arandu
x = 10
{
    x = 20 // Warning: shadows outer x
}
```

## Semantic Explanation

Shadowing can cause confusion and accidental bugs when programmers read or modify the code expecting the outer variable.

## How to Fix

Choose a unique name or suppress shadowing warnings locally with `@Suppress("shadowing")`:

```arandu
x = 10
{
    inner_x = 20
}
```
