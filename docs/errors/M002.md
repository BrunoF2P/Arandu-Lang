# Error M002: Undefined Namespace Member

This error occurs when trying to access a member inside a module namespace that does not exist.

## Incorrect Code Example

```arandu
import io
func main() {
    io.non_existent()
}
```

## Semantic Explanation

The compiler verifies namespace member accesses against the public symbols exported by the target module. If the symbol is missing, it fails name resolution.

## How to Fix

Use a valid public member exported by the module:

```arandu
import io
func main() {
    io.println("Hello")
}
```
