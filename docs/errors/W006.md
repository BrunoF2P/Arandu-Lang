# Error W006: Unhandled Result

This warning occurs when a function returns a `Result` type but the caller discards it without checking or propagating.

## Incorrect Code Example

```arandu
func save() -> Result[void, str] {}
func main() {
    save() // Warning: Result ignored
}
```

## Semantic Explanation

Discarding `Result` values silently ignores potential errors, making applications fragile and hard to debug.

## How to Fix

Check the result or propagate it using the try operator `?`:

```arandu
func save() -> Result[void, str] {}
func main() {
    save()?
}
```
