# Error T005: Operator Not Applicable

This error occurs when applying an operator (mathematical, logical, or comparison) to types that do not support it.

## Incorrect Code Example

```arandu
x = "hello" + 5
```

## Semantic Explanation

Operators are defined only for specific types (e.g. arithmetic for `int`, logical for `bool`). String concatenation using `+` is not supported; use standard library methods.

## How to Fix

Use compatible types with the operator:

```arandu
x = 5 + 5
```
