# Error N010: Undefined Associated Function

This error occurs when calling an associated function on a type namespace that does not exist.

## Incorrect Code Example

```arandu
struct User {}
func main(): {
    User.missing()
}
```

## Semantic Explanation

Associated functions must be explicitly declared on the type namespace using the `Type.func_name` syntax.

## How to Fix

Declare the associated function:

```arandu
struct User {}
func User.new() -> User: {
    return User {}
}
func main() {
    u = User.new()
}
```
