# Error N002: Undefined Type

This error occurs when referencing a type name that is not declared or visible in the current scope.

## Incorrect Code Example

```arandu
func main(p: Person) {}
```

## Semantic Explanation

Type annotations must reference valid primitive types (e.g., `int`, `str`) or named types (e.g., `struct`, `interface`, `enum`) that have been defined.

## How to Fix

Define the type or check the spelling:

```arandu
struct Person {
    name: str
}
func main(p: Person) {}
```
