# Error O012: `alloc` Requires `unsafe`

Heap allocation via language `alloc` (or equivalent unsafe allocation forms)
must appear inside an `unsafe` block. This keeps raw allocation intentional and
auditable.

## Incorrect Code Example

```arandu
func main() {
    let p = alloc int
}
```

## Correct Code Example

```arandu
func main() {
    unsafe {
        let p = alloc int
        free p
    }
}
```
