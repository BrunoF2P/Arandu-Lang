# Warning W007: Unused Import

This warning occurs when a module or item is imported into a scope using `import`, but is never referenced or consumed anywhere in the file.

## Incorrect Code Example

```arandu
module app.main

import std.io as io // Warning W007: unused import `io`

func main() {
    let x = 42
}
```

## Semantic Explanation

Importing unused modules or symbols increases compilation scope overhead and clutters the top-level namespace. Unlike strict languages that fail compilation on unused imports, Arandu Lang emits a non-blocking warning (`W007`) to preserve fast prototyping velocity while encouraging clean code hygiene.

## How to Fix

Either consume the imported item in your program, or remove the unused `import` statement:

```arandu
module app.main

func main() {
    let x = 42
}
```
