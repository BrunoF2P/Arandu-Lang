# Error P005: Expected Expression

This error occurs when the parser expects an expression (such as a value, variable, or function call) but encounters a token that cannot start an expression.

## Incorrect Code Example

```arandu
x = *
```

## Semantic Explanation

An expression must begin with a valid term (like a literal, identifier, prefix operator, or opening parenthesis). Binary operators like `*` cannot start an expression.

## How to Fix

Provide a valid expression:

```arandu
x = 5 * 2
```
