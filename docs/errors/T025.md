# Error T025: Interface Not Satisfied

This error occurs when assigning a type instance to an interface variable but the type does not implement the interface.

## Incorrect Code Example

```arandu
interface Runnable {
    func run() void
}
struct User {}

func main() {
    r Runnable = User {}
}
```

## Semantic Explanation

Variables declared with interface types can only hold instances of types that explicitly implement all methods required by the interface.

## How to Fix

Implement the interface methods on the type:

```arandu
interface Runnable {
    func run() void
}
struct User {}
func User.run(self) {}

func main() {
    r Runnable = User {}
}
```
