# Error O011: Free Requires Pointer Type

This error occurs when using the `free` statement on a type that is not a raw pointer.

## Incorrect Code Example

```arandu
let x int = 5
free x
```

## Semantic Explanation

The `free` statement reclaims raw memory allocations and must target raw pointers (e.g. `ptr[T]`).

## How to Fix

Apply `free` only to pointer types:

```arandu
let x ptr[int] = alloc 5
free x
```
