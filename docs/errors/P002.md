# Error P002: Unclosed Block

This error occurs when a block of code (such as a function or statement block) is opened with '{' but the file ends before it is closed.

## Incorrect Code Example

```arandu
func main() {
    x = 10
// EOF
```

## Semantic Explanation

Every opened block scope must be balanced by a matching closing brace. The parser keeps track of block nesting and reports an error if EOF is reached with open scopes.

## How to Fix

Add the missing closing brace to terminate the block scope:

```arandu
func main() {
    x = 10
}
```
