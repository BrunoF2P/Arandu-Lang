# Error M003: Namespace Used as a Value

This error occurs when a module namespace is evaluated directly as a value.

## Incorrect Code Example

```arandu
import io
func main() {
    x = io
}
```

## Semantic Explanation

Namespaces/modules represent scopes and cannot be stored, passed, or manipulated as runtime values.

## How to Fix

Access a member of the namespace or reference a value instead:

```arandu
import io
func main() {
    x = io.println
}
```
