# Error T001: Cannot Infer Type

This error occurs when the type checker cannot determine the type of a variable without a type annotation.

## Incorrect Code Example

```arandu
func main() {
    let x = null
}
```

## Semantic Explanation

Arandu uses type inference, but some expressions (like a bare `null` or uninitialized variables) do not provide enough type context.

## How to Fix

Add an explicit type annotation:

```arandu
func main() {
    let x ptr[int] = null
}
```
