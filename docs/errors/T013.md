# Error T013: Unknown Named Argument

This error occurs when calling a function using a named argument that does not match any parameters.

## Incorrect Code Example

```arandu
func greet(name: str) {}
func main() {
    greet(age: 30)
}
```

## Semantic Explanation

Named argument parameter passing must target valid parameter names declared in the function signature.

## How to Fix

Use the correct parameter name:

```arandu
func greet(name: str) {}
func main() {
    greet(name: "Bruno")
}
```
