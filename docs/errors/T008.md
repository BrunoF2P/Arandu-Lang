# Error T008: Match Arm Mismatch

This error occurs when different arms of a `match` expression evaluate to incompatible types.

## Incorrect Code Example

```arandu
x = match val {
    1 => 5,
    2 => "hello"
}
```

## Semantic Explanation

Like if-expressions, match-expressions must yield a single, unified type across all possible execution branches.

## How to Fix

Make all arms return the same type:

```arandu
x = match val {
    1 => 5,
    2 => 10
}
```
