# Error N003: Redefined Name

This error occurs when declaring a variable, function, or type name that is already defined in the same scope block.

## Incorrect Code Example

```arandu
func main() {
    x = 10
    x = 20
}
```

## Semantic Explanation

Each scope block cannot have duplicate definitions for the same identifier to avoid ambiguity.

## How to Fix

Use a different name or mutate the existing variable instead of redeclaring it:

```arandu
func main() {
    mut x = 10
    set x = 20
}
```
