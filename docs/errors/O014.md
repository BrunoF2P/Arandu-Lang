# Error O014: `free` Requires `unsafe`

Deallocation with `free` is an unsafe operation and must live inside an
`unsafe` block (paired with an earlier `alloc` or equivalent).

## Incorrect Code Example

```arandu
func main() {
    unsafe {
        let p = alloc int
    }
    free p
}
```

## Correct Code Example

```arandu
func main() {
    unsafe {
        let p = alloc int
        free p
    }
}
```
