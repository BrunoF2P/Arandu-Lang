# Error T033: Indirect Call Not Supported

Calling through a non-direct callee (function pointer, abstract method
dispatch via a type parameter, or other indirect call form) is not supported
in the current compiler profile. Prefer a direct, monomorphizable call.

## Incorrect Code Example

```arandu
interface I {
    func m(shared self): int
}

func call<T: I>(shared x: T): int {
    // Method dispatch through a type parameter is not a direct call yet.
    return x.m()
}
```

## Correct Code Example

```arandu
// Use a concrete type and a direct method or free function.
struct Box {
    n: int
}

func Box.m(shared self): int {
    return self.n
}

func call(shared x: Box): int {
    return x.m()
}
```

## Notes

- This is the Minimal / gold-suite fence for interface methods via `T: I`.
- Free-function generics that monomorphize to a direct call remain allowed.
- See `PROMOTE-L1` in the Minimal freeze doc for the promotion track.
