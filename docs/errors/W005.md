# Error W005: Unnecessary Mutability

This warning occurs when a variable is declared with mutability but is never reassigned or mutated.

## Incorrect Code Example

```arandu
let mut x = 10 // Warning: x does not need to be mutable
```

## Semantic Explanation

Variables should be immutable by default. Mutability should only be used when value changes are required.

## How to Fix

Remove the mutability modifier:

```arandu
let x = 10
```
