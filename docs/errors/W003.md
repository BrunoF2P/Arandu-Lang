# Error W003: Unreachable Code

This warning occurs when statements are placed after control flow instructions that terminate execution unconditionally (like return or break).

## Incorrect Code Example

```arandu
func main() {
    return
    x = 10 // Warning: unreachable
}
```

## Semantic Explanation

Unreachable code is redundant and usually indicates logical flaws in control flow design.

## How to Fix

Remove the unreachable statements or fix the control flow.
