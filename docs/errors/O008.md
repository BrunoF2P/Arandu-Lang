# Error O008: Use of Uninitialized Variable

This error occurs when attempting to read a variable before it is initialized in all execution paths.

## Incorrect Code Example

```arandu
x: int
y = x + 5
```

## Semantic Explanation

Variables must contain a valid value before they are read. Static analysis verifies that an assignment occurs on every path leading to the read.

## How to Fix

Initialize the variable during declaration:

```arandu
x: int = 10
y = x + 5
```
