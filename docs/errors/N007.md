# Error N007: Undefined Assignment Target

This error occurs when trying to assign a value using `set` to a target that cannot be resolved or written to.

## Incorrect Code Example

```arandu
func main() {
    set x = 10
}
```

## Semantic Explanation

The `set` statement expects a mutable, declared variable or field. Assigning to an undeclared target fails name resolution.

## How to Fix

Declare the variable with `let` before mutating it with `set`:

```arandu
func main() {
    let x = 5
    set x = 10
}
```
