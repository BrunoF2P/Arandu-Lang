# Error O013: `extern` Body Requires `unsafe`

Calling or defining FFI/`extern` operations that touch the unsafe world must
be nested in `unsafe` so the boundary is explicit.

## Incorrect Code Example

```arandu
extern "C" {
    func c_puts(s: str): int
}

func main(): int {
    return c_puts("hi")
}
```

## Correct Code Example

```arandu
extern "C" {
    func c_puts(s: str): int
}

func main(): int {
    unsafe {
        return c_puts("hi")
    }
}
```
