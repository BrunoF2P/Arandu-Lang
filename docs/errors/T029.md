# Error T029: Recursive type has infinite size

A struct directly or indirectly contains itself as a field without an indirection pointer. This results in a type of infinite size, which cannot be laid out in memory at compile time.

## Erroneous Code Example

```arandu
struct ListNode {
    val: i32,
    next: ListNode, // error: recursive type 'ListNode' has infinite: size
}
```

## Semantic Explanation

To calculate the size of a struct, the compiler must recursively sum the sizes of all its fields. If a struct contains itself directly (or through a cycle of other structs), calculating its size requires resolving an infinite recursion. To break this infinite recursion and define a concrete size, you must use indirection (such as a pointer type `ptr[T]`). Pointers have a fixed, well-known size.

## How to Fix

Wrap the recursive reference inside a pointer type:

```arandu
struct ListNode {
    val: i32,
    next: Option[ptr[ListNode]], //: ok!
}
```
