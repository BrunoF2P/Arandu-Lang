# Error T014: Invalid Variadic Type

This error occurs when passing an invalid argument to a variadic function.

## Incorrect Code Example

```arandu
func print_all(args ...int) {}
func main() {
    print_all("hello")
}
```

## Semantic Explanation

Variadic arguments represent a homogeneous list of values. Arguments passed must unify with the type of the variadic parameter.

## How to Fix

Pass values matching the variadic type:

```arandu
func print_all(args ...int) {}
func main() {
    print_all(1, 2, 3)
}
```
