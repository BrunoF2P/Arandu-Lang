# Error T007: If/Else Branch Mismatch

This error occurs when the `if` and `else` branches of a conditional expression evaluate to incompatible types.

## Incorrect Code Example

```arandu
x = if true { 5 } else { "hello" }
```

## Semantic Explanation

If-expressions return a value. Both the `then` and `else` branches must evaluate to the same type so the overall expression has a single type.

## How to Fix

Ensure branches return matching types:

```arandu
x = if true { 5 } else { 10 }
```
