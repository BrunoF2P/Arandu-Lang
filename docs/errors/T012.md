# Error T012: Wrong Argument Count

This error occurs when calling a function or constructor with a different number of arguments than declared.

## Incorrect Code Example

```arandu
func add(a: int, b: int) -> int { return a + b }
func main() {
    add(5)
}
```

## Semantic Explanation

Function calls must provide exactly the number of arguments matching the function parameter count, unless the function is variadic.

## How to Fix

Provide the correct number of arguments:

```arandu
func add(a: int, b: int) -> int { return a + b }
func main() {
    add(5, 10)
}
```
