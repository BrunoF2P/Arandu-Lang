# Error P003: Invalid Assignment Operator

This error occurs when an invalid symbol or combination of symbols is used as an assignment operator.

## Incorrect Code Example

```arandu
set x =+ 10
```

## Semantic Explanation

Arandu supports specific assignment operators (e.g., `=`, `+=`, `-=`, `*=`, `/=`). Other combinations like `=+` are parsed as invalid.

## How to Fix

Use a valid assignment operator:

```arandu
set x += 10
```
