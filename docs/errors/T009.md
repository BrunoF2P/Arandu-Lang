# Error T009: Condition Not a Boolean

This error occurs when the condition of an `if` or `while` statement is not a boolean type.

## Incorrect Code Example

```arandu
if 10 {
}
```

## Semantic Explanation

Control flow conditions must evaluate strictly to a boolean value (`true` or `false`). Implicit truthiness conversion of integers or strings is not allowed.

## How to Fix

Use a comparison or boolean value:

```arandu
if 10 > 5 {
}
```
