# Error T028: Field initialized more than once

A field in a struct initializer literal was initialized more than once.

## Erroneous Code Example

```arandu
struct Point {
    x: i32,
    y: i32,
}

fn main() {
    let p = Point {
        x: 10,
        y: 20,
        x: 30, // error: field 'x' initialized more than once
    };
}
```

## Semantic Explanation

Each field in a struct literal must be initialized exactly once. Providing multiple initializations for the same field is redundant and creates ambiguity regarding which value should be assigned.

## How to Fix

Remove the duplicate field initialization:

```arandu
struct Point {
    x: i32,
    y: i32,
}

fn main() {
    let p = Point {
        x: 10,
        y: 20,
    }; // ok!
}
```
