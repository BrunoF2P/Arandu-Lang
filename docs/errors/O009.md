# Error O009: Lifetime Mismatch

This error occurs when the lifetimes of references in arguments or assignments do not unify correctly.

## Incorrect Code Example

```arandu
// Lifetimes mismatch in generic references
```

## Semantic Explanation

Lifetimes prevent dangling references. If a struct expects a reference with a longer lifetime but you provide a shorter one, lifetimes mismatch.

## How to Fix

Ensure lifetimes of reference parameters are compatible.
