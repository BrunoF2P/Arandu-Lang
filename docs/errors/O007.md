# Error O007: Inconsistent Move Between Branches

This error occurs when a variable is moved in one branch of a conditional block (like if/else) but not in the other, leaving its state ambiguous.

## Incorrect Code Example

```arandu
let x = Data {}
if condition {
    process(x) // Moved
} else {
    // Not moved
}
let y = x // Ambiguous: x might be moved here!
```

## Semantic Explanation

The compiler requires deterministic knowledge of variable initialization and move status at every instruction. Ambiguous move states are rejected.

## How to Fix

Ensure the variable is moved in both branches, or not moved in either branch, or handle it inside the conditional block:

```arandu
let x = Data {}
if condition {
    process(x)
} else {
    process(x)
}
```
