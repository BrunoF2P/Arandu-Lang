# Error T004: Incompatible Return Type

This error occurs when a function returns a value whose type does not match its declared return type.

## Incorrect Code Example

```arandu
func main() -> int {
    return "hello"
}
```

## Semantic Explanation

The type of the expression returned must match the return type specified in the function signature.

## How to Fix

Return the correct type or modify the signature:

```arandu
func main() -> str {
    return "hello"
}
```
