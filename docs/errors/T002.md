# Error T002: Incompatible Assignment

This error occurs when attempting to assign a value of one type to a variable declared with a different type.

## Incorrect Code Example

```arandu
func main() {
    x: int = "hello"
}
```

## Semantic Explanation

Arandu is statically typed. The type of the value on the right-hand side must match the variable type on the left-hand side.

## How to Fix

Ensure the types match:

```arandu
func main() {
    x: str = "hello"
}
```
