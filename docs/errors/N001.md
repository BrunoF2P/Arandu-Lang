# Error N001: Undefined Value

This error occurs when referencing a variable or function name that has not been defined in the current or outer scopes.

## Incorrect Code Example

```arandu
func main() {
    x = y + 10
}
```

## Semantic Explanation

Before check time, Name Resolution creates a scope tree and resolves identifier names to symbol declarations. Unresolvable names trigger undefined value.

## How to Fix

Declare the variable or import the function before using it:

```arandu
func main() {
    y = 5
    x = y + 10
}
```
