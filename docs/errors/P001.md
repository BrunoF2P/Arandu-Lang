# Error P001: Unexpected Token

This error occurs when the parser encounters a token that does not match the grammatical rules at that position.

## Incorrect Code Example

```arandu
func main() {
    x = 10 +
}
```

## Semantic Explanation

The parser matches tokens against the grammar rules using Pratt parsing for expressions. If a token appears out of order (such as a closing brace directly after an addition operator), parsing fails.

## How to Fix

Ensure your syntax matches the language grammar:

```arandu
func main() {
    x = 10 + 5
}
```
