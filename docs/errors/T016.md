# Error T016: Invalid Try Operator

This error occurs when applying the `?` operator to an expression that is not a `Result` or `Option` type.

## Incorrect Code Example

```arandu
func main() {
    x = 10
    y = x?
}
```

## Semantic Explanation

The try operator `?` is syntactic sugar to check and propagate errors/nils. It can only be used on `Result` and `Option` types.

## How to Fix

Apply `?` only to Result or Option values:

```arandu
func perform() -> Result[int, str] {
    return Result.Ok(10)
}
func main() {
    r = perform()?
}
```
