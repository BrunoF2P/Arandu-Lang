# Error P006: Malformed Attribute

This error occurs when an attribute or annotation is malformed or has invalid arguments.

## Incorrect Code Example

```arandu
@Suppress()
func main() {}
```

## Semantic Explanation

Attributes like `@Suppress` must follow the correct syntax, containing one or more string literals designating lints to be suppressed.

## How to Fix

Provide correct arguments to the attribute:

```arandu
@Suppress("unused")
func main() {}
```
