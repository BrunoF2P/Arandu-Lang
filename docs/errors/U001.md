# Error U001: Feature Not Supported

This error occurs when trying to use a language feature that is defined in the syntax but not yet implemented in the compiler version.

## Incorrect Code Example

```arandu
// Using unimplemented compiler feature
```

## Semantic Explanation

The syntax is recognized by the parser, but the current backend/lowerings do not support code generation for it.

## How to Fix

See docs/arandu-compiler-roadmap-v0.1.md to track feature implementation status.
