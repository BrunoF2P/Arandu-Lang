# Error T003: Incompatible Argument

This error occurs when passing an argument type that does not match the parameter type of the called function.

## Incorrect Code Example

```arandu
func greet(name: str) {}
func main() {
    greet(42)
}
```

## Semantic Explanation

Functions have strict type signatures. The types of arguments at the call site must unify with the parameter types in the function definition.

## How to Fix

Pass the expected type:

```arandu
func greet(name: str) {}
func main() {
    greet("Bruno")
}
```
