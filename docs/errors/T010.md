# Error T010: Invalid Cast

This error occurs when attempting an explicit type cast that is invalid or not supported.

## Incorrect Code Example

```arandu
x = "hello" as int
```

## Semantic Explanation

Type casting (`as`) is only allowed between compatible types (e.g., numeric types, pointers). Direct cast from strings to numbers is not allowed.

## How to Fix

Use parsing functions for strings or correct the cast type:

```arandu
x = 5.5 as int // Casts float to int
```
