# Error T017: Invalid Index

This error occurs when trying to index an array or collection with a non-integer index type.

## Incorrect Code Example

```arandu
x = [1, 2, 3]
y = x["hello"]
```

## Semantic Explanation

Array and collection indices must evaluate to integer types representing the item position.

## How to Fix

Use an integer value as the index:

```arandu
x = [1, 2, 3]
y = x[0]
```
